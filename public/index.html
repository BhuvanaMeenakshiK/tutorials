Neutrino Profanity Filter

Questionable content

Pressing this button will publish a message to a channel, which will in turn using an Ably Reactor Queue rule to republish the message into a queue. The Node.js worker server will then consume this message in real time, send the text to Neutrino using its Bad Words REST API, and publish the text without bad words back to the client using an Ably channel.
